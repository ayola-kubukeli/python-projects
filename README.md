# Python Projects

This repository contains a few beginner Python projects I built while learning programming.

The goal of these projects is to practice coding basics and understand how real applications work behind the scenes.

---

## Projects Included

### Expense Tracker

A simple command-line program that lets a user add and view their expenses.

It saves data to a file so that information is not lost when the program closes.

What I practiced:
- Working with files
- Using functions
- Handling lists and dictionaries
- Building a simple menu system

---

### Crypto Price Checker

A small Python program that fetches live cryptocurrency prices using an API.

It takes data from the internet and displays it in a readable format.

What I practiced:
- Working with APIs
- Handling JSON data
- Processing and displaying data
- Understanding how external services connect to programs

---

## What I learned from these projects

- How to break problems into smaller parts
- How data is stored and retrieved in simple applications
- How APIs work in real-world systems
- How to structure basic Python programs
